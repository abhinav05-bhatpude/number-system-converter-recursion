# 🔢 Number System Converter (Recursion)

## 📌 Overview
This project is a number system converter built using JavaScript recursion.  
It converts decimal numbers into binary, octal, and hexadecimal formats.

---

## 🚀 Features (Day 1)
- Decimal to Binary (Recursion)
- Decimal to Octal (Recursion)
- Decimal to Hexadecimal (Recursion)

- Problem breakdown
- Mathematical logic
